# Alias, c'est quoi ?

!!BROUILLON A ECRIRE!!

Alias est une API pour les devs et une interface pour les DPO Utilité : assurer une cohérence et une véracité juridique et technique vis à vis du RGPD. Gerer la conformité RGPD des données depuis les docs d'accountability jusqu'à l'implémentation technique pour les devs.

Avantages ? Pourquoi Alias ?

Elle permet d'assurer un lien entre DPO et équipes techniques. Vision à jour et à 360° des données persos au sein d'un système d'information.

Automatiser le contrôle du respect du RGPD, centraliser la gestion de tous les consentements en temps réel.

Cartographier les données, où qu'elles soient stockées (DB, Bucket, document papier, SASS, etc.)

Permet de répondre aux requêtes de droits RGPD (accès, portabilité, suppression, modification, accès) en quelques minutes.

Confidentialité: nous ne stockons ni n'accédons à aucune de vos données.
